# 📱 KNN Mobile Price Classifier

This is a beginner-friendly Machine Learning project that classifies mobile phones into **Low 💸**, **Medium 💰**, and **High 💎** price ranges using the **K-Nearest Neighbors (KNN)** algorithm.

---

## 📌 Features
- Classifies mobile phones based on specifications
- Uses RAM, Storage, and Battery as input features
- Simple and easy to understand ML project
- Real-time prediction using user input

---

## 🧠 Algorithm Used
**K-Nearest Neighbors (KNN)**  
- Finds the nearest data points  
- Predicts based on majority voting  

---

## 📊 Dataset
The dataset contains:
- `ram` (GB)
- `storage` (GB)
- `battery` (mAh)
- `price_range`  
  - 0 = Low 💸  
  - 1 = Medium 💰  
  - 2 = High 💎
